#  Sales Data Analysis – 
##  Project Description
This project delivers an end-to-end Sales Data Analysis solution using Power BI Desktop, with a strong focus on data modeling concepts. The report is designed based on defined business requirements and provides insights into sales performance, product trends, and profitability. It demonstrates hands-on experience with data transformation, star schema modeling, relationships, DAX calculations, and advanced filtering techniques.

##  Project Implementation Steps
1️⃣ Business Understanding
•	Analysed and documented business requirements
•	Identified key metrics such as:
o	Sales
o	Quantity
o	Profit
o	Time-based trends

2️⃣ Data Loading & Preparation
•	Loaded sales data into Power BI Desktop
•	Performed data profiling
•	Applied data transformations in Power Query (Part 1 & Part 2)
•	Cleaned and structured data for modeling

3️⃣ Data Modeling Concepts
•	Defined Primary and Foreign Keys
•	Established relationships with correct cardinality
•	Implemented a Star Schema model
•	Reviewed overall data model structure to ensure optimal performance

4️⃣ Filters & Interactions
•	Explored different filter types in the Filters pane:
o	Visual-level
o	Page-level
o	Report-level
•	Customized filter behavior for dimension table slicers
•	Edited visual interactions for better report usability

5️⃣ Sales Analysis & DAX Measures
•	Created Top & Bottom 5 Products
o	By Sales
o	By Quantity
o	By Profit
•	Built Sales Trend analysis over time
•	Implemented multiple DAX measures to meet business requirements

6️⃣ Report Enhancements
•	Addressed additional business requirements
•	Optimized report layout and user interactions
•	Ensured accurate filtering and cross-highlighting behavior

##  Key Insights Delivered
•	Identification of top and underperforming products
•	Sales and profit trends across time
•	Clear understanding of product performance drivers
•	Optimized data model for scalable reporting

##  Conclusion
This project showcases strong expertise in Power BI data modeling, DAX, and business-driven analytics. It highlights the ability to translate business requirements into a well-structured star schema model and interactive sales dashboards that support informed decision-making.
